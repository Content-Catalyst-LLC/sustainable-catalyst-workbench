# Sustainable Catalyst Workbench v3.0.0

## Unified Prototyping Workbench

Version 3.0.0 consolidates the v2.x studio sequence into a shared production environment without removing specialist interfaces.

### New capabilities

- Unified Project Hub and canonical project manifest
- Studio registry, dependency review, and project-health scoring
- Shared evidence and artifact inventory
- Site Intelligence, Decision Studio, Research Librarian, and Workbench handoff packets
- Versioned project packages with SHA-256 hashes and chain linking
- Workspace-health and storage-integrity records
- Selective, category, and complete reset planning
- Mandatory backup and confirmation safeguards for destructive actions
- v2.x browser-storage migration planning without automatic deletion
- Unified local-runner tool discovery
- Primary `[sc_workbench]` shortcode defaults to the unified hub while retaining all specialist studios

### Compatibility

All v2.0.0–v2.9.0 shortcodes and modules remain available. Existing browser records are not deleted or silently migrated. Version 3.0.0 generates a migration plan and requires an explicit backup before reset operations.
