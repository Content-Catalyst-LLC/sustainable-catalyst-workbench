# Workbench v5.0.0 Integrated Platform

The Integrated Platform is the canonical coordination layer for Sustainable Catalyst research and engineering projects. It does not replace specialist studios. It connects their records through versioned, content-hashed contracts.

## Canonical records

- Integrated project
- Surface registry
- Project portfolio
- Cross-surface workflow
- Integrity audit
- Governance gate
- Deployment plan
- Integrated dossier
- Portable platform package

## Connected surfaces

Workbench, Lab, Site Intelligence, Decision Studio, Research Librarian, Knowledge Library, private WordPress storage, and offline Workbench nodes.

## Release boundary

A project cannot reach an approved-for-human-controlled-release state without integrity, traceability, security, evaluation, professional review when high stakes, resolved critical findings, and a named human approver. Approval never authorizes automatic publication or certification.
