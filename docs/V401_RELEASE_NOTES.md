# Workbench v4.0.1 — Connected Environment Activation and Integration Reliability

Version 4.0.1 stabilizes the v4 connected milestone without adding another unified studio. It adds a dedicated reliability console, live activation auditing, schema compatibility checks, active-project propagation tests, cross-platform handoff round-trip validation, asset/cache auditing, offline fallback planning, safe repair plans, and end-to-end Sustainable Catalyst fixtures.

The patch preserves all 22 v4.0.0 studios and every existing shortcode. Repair remains human-controlled: no destructive migration, automatic publication, remote shell, automatic device control, or automatic deletion is authorized.
