# Workbench v4.3.0 — Live Data Connectors and Reproducible Dataset Pipelines

v4.3.0 adds private source registries, connector planning and health records, content-hashed dataset manifests, dependency-aware transformation pipelines, validation, freshness and cache plans, offline snapshots, provenance records, refresh orchestration, and portable reproducible dataset packages.

The release adds the 25th unified studio, **Data Pipelines**, while preserving browser-local, WordPress, and offline operation.
