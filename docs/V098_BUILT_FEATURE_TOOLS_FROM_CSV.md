# Sustainable Catalyst Workbench v0.9.8 — Built Feature Tools

This release uses the uploaded Feature Builder CSV as an implementation queue and builds all 59 listed features into the Workbench tool registry.

## What is included

- 59 backend tool IDs generated from the Feature Builder CSV.
- 59 WordPress local calculator definitions, so tools appear in the Calculate tab even before the backend response returns.
- Deterministic MVP engines for each feature family.
- Built feature tools CSV and JSON manifest bundled in the plugin.
- Professional-review warnings for engineering, clinical, structural, RF/power, materials, lab, aerospace, and safety-critical areas.

## Major feature families

- Feature builder / equation registry infrastructure
- Weighted composite indicators
- Forecasting and time-series diagnostics
- Stock-flow, recurrence, ODE, and limits-to-growth dynamics
- Risk, resilience, policy coherence, economics, and planetary-pressure tools
- Optimization, linear algebra, causal inference, Bayesian updating, networks, and algorithmic complexity
- Physics, chemistry, biology/ecology, climate, materials, electrical, RF, power, structural, lab, clinical, and aerospace calculators
- Content frameworks, behavioral systems, psychology profiles, narrative, symbolism, and meaning matrices

## Important boundary

v0.9.8 implements broad MVP calculators from equation-registry signals. These tools are not certified design tools, clinical tools, legal/financial advice, or safety-critical systems. Treat them as transparent analytical scaffolds for article-aware learning, prototyping, and feature development.
