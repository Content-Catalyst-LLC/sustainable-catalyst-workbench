# Workbench v3.3.0 Security Boundary

- Public visitors operate browser-locally.
- WordPress evidence and handoff records are private and owner-scoped.
- Handoff packets are data records, not executable commands.
- Portable bundles include attachment manifests, not unrestricted file execution.
- Receiving applications must verify hashes and preserve provenance.
- Human review is required before publication, execution, certification, or policy use.
