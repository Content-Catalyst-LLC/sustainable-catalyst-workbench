# Workbench v4.2.0 Security and Responsibility Boundary

- Templates and scaffolds are planning records, not execution authority.
- Automatic experiment execution, device control, publication, and certification are disabled.
- Starter evidence records are placeholders and cannot support claims until verified.
- Private WordPress template and guided-project records require authenticated editing capability.
- Browser-local and offline operation remain available.
- Export packages remove common secret-bearing fields and require explicit import.
- High-stakes projects require qualified human review.
