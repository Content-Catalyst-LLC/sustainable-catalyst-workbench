# Workbench v3.5.0 Device-Control Security Boundary

Workbench v3.5.0 produces explicit allowlisted plans and auditable records. It does not expose arbitrary shell execution, remote shells, safety-control bypasses, or unsupervised device control. Sensitive operations require fresh local consent and a confirmation phrase. Simulation twins are clearly labeled and never represented as physical measurements. Device use remains subject to manufacturer instructions, laboratory procedures, electrical and machinery safety, applicable law, and qualified human supervision.
