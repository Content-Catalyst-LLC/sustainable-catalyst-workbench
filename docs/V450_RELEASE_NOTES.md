# Workbench v4.5.0 — Extension SDK, Plugin Registry, and Third-Party Module Framework

- Adds a 27th unified studio: Extension SDK.
- Adds versioned extension manifests and semantic core compatibility checks.
- Adds private registry and extension-release records.
- Adds least-privilege capability and host allowlist audits.
- Adds lifecycle, hook, scaffold, security-audit, and package contracts.
- Preserves browser-local, authenticated WordPress, and offline development modes.
- Blocks arbitrary code execution, remote shells, privilege escalation, automatic installation, automatic activation, device execution, publication, and certification.
