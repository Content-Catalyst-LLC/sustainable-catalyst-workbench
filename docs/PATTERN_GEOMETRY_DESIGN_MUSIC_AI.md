# Pattern, Geometry, Design, Music, and AI Workbench Cluster

Version 0.6.0 adds a distinct Workbench family for pattern systems across mathematics, music, color, vector geometry, AI representation, and multimodal analysis.

## Included tools

- Music Frequency Calculator
- Chord and Scale Identifier
- Color Contrast and Luminance Calculator
- Color Harmony Generator
- Vector Geometry Calculator
- Embedding Similarity Tool
- PCA / Dimensionality Reduction Explorer
- Fourier / Frequency Analysis Tool
- AI Classification Metrics Calculator
- Multimodal Pattern Comparison Tool

## Core formulas

- MIDI to frequency: `f = 440 * 2^((m - 69) / 12)`
- Cents: `c = 1200 * log2(f2 / f1)`
- Contrast ratio: `(L1 + 0.05) / (L2 + 0.05)`
- Dot product: `a · b = Σ a_i b_i`
- Cosine similarity: `(a · b) / (||a|| ||b||)`
- PCA: SVD of centered matrix `X = UΣVᵀ`
- Classification F1: `2PR / (P + R)`

These tools are intended to connect mathematics, design, music, color, AI embeddings, visualization, and meaning without turning the Research Library into a large tool drawer.
