# Sustainable Catalyst Workbench v3.5.0

## Advanced Device and Instrument Orchestration

- Connected-device inventory and capability discovery
- Explicit local consent, operation allowlists, and expiration
- Sensitive-action confirmation for flash, write, reset, upload, and power-cycle operations
- Instrument sessions and calibration schedules
- Ordered experiment-run plans with rollback metadata
- Chained device logs and failure-recovery plans
- Clearly labeled deterministic hardware-absent simulation twins
- Optional private WordPress device and run records
- No arbitrary shell or remote-command execution

The release preserves all earlier Workbench studios and adds Device Orchestration as the seventeenth unified studio.
