GITHUB UPGRADE

cd ~/Downloads
unzip sustainable-catalyst-workbench-v2.2.0-upgrade-bundle.zip
cd sustainable-catalyst-workbench-v2.2.0-upgrade-bundle
chmod +x push_workbench_v2_2_0_to_github.sh
./push_workbench_v2_2_0_to_github.sh

The script applies v2.0.0 and v2.1.0 prerequisites when needed, applies v2.2.0, normalizes source whitespace, runs release checks, creates the complete WordPress plugin and repository ZIPs, and pushes main.
