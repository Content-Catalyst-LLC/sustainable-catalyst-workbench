# Workbench v3.9.0 Security Boundary

- Production evaluation records are plans and evidence, not certifications.
- Public release is never automatic.
- Release gates require explicit human approval.
- Unauthenticated public write routes are blocking findings.
- Local services remain loopback-only by default.
- Arbitrary command execution and remote shells remain prohibited.
- Critical and high unresolved findings block readiness.
- Evaluation records are private when stored in WordPress.
