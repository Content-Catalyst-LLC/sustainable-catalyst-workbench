# Workbench v4.4.0 security and evaluation boundary

- Benchmark and evaluation-run records are private by default.
- No arbitrary command execution or remote shell is added.
- Experiment matrices are plans and never execute automatically.
- Leaderboards remain provisional and cannot publish automatically.
- Baseline replacement, release approval, and certification require explicit human review.
- High-stakes evaluations require qualified domain and governance review.
- Portable packages require explicit import and preserve component hashes.
