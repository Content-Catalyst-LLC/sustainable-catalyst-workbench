# Workbench v3.9.0 Public Release Checklist

## Required evaluation records

- Accessibility report
- Performance budget report
- Security and privacy posture report
- Compatibility matrix
- Migration and storage stress report
- Failure-injection and recovery report
- Onboarding package
- Stable extension contract

## Required release controls

- No unresolved high or critical findings
- Package checksums verified
- Backup and rollback verified
- Public documentation complete
- Human reviewer explicitly approves the gate
- Automatic publication remains disabled

## Evidence boundary

The checklist records the evidence reviewed for a release decision. It is not a
regulatory certification and does not replace qualified professional review.
