# Sustainable Catalyst Prototyping Workbench v2.8.0

## Experiment Automation and Reproducible Workflow Studio

Version 2.8.0 turns the Workbench's instruments, simulations, runtimes, and visualizations into structured and auditable experiment workflows.

### New studios

- Experiment Automation Studio
- Structured Protocol Builder
- Reproducible Workflow Pipeline Studio
- Experiment Schedule and Local Task Planner
- Dataset, Configuration, and Checkpoint Studio
- Reproducibility and Deviation Audit Studio

### Core capabilities

- Dependency-aware protocol and task validation
- Cycle, unknown-dependency, duplicate-ID, and resource-conflict detection
- Critical-path and resource-use estimates
- Repeatable schedule plans without silently installing unattended operating-system jobs
- Canonical SHA-256 dataset, configuration, environment, workflow, and bundle manifests
- Automatic numeric, equality, range, and finite-value checkpoints
- Cross-run dataset, configuration, code-revision, and metric comparison
- Standard deviation, coefficient of variation, absolute delta, and relative delta records
- Explicit failure and deviation logs
- Browser-local save and exportable reproducible experiment bundles
- Paired local discovery for Python, pytest, Git, Make, Jupyter, Papermill, Snakemake, Nextflow, DVC, and SQLite

### Compatibility

The v2.8.0 release preserves v2.0.0-v2.7.0 shortcodes, REST routes, runner pairing, browser storage, and project records.

### Boundaries

This release does not authorize unattended physical experiments, bypass equipment interlocks, install a system scheduler, or replace laboratory supervision, quality systems, safety review, or professional validation.
