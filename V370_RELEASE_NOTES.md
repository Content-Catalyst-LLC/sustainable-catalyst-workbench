# Sustainable Catalyst Workbench v3.7.0

## Domain Laboratory Integration

This release connects Workbench projects to Sustainable Catalyst Lab domain workspaces through shared, content-hashed contracts for domain profiles, calculations, experiments, validation, notebook entries, reports, cross-language equivalence, portable bundles, and synchronization plans.

The integration layer covers active and planned scientific, engineering, urban, economic, aerospace, biological, biomedical, and health/human-services domains. It does not authorize physical execution, clinical diagnosis, or licensed certification.
