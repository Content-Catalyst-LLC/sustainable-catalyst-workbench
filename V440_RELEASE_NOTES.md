# Workbench v4.4.0 — Automated Evaluation, Benchmarking, and Comparison Laboratory

## Summary

Adds auditable benchmark definitions, experiment matrices, repeated-trial summaries, baseline/candidate comparisons, regression detection, reproducibility checks, provisional leaderboards, evaluation gates, and portable evaluation packages.

## New unified studio

- Evaluation Lab (`sc_workbench_evaluation_lab`)

## Private WordPress records

- `scwb_benchmark`
- `scwb_eval_run`

## Safety boundary

The release does not automatically execute experiments, publish rankings, replace baselines, approve releases, establish causation, or certify professional conclusions.
