# Workbench v4.1.0 — Hosted Collaborative Workspace and Authenticated Team Projects

Version 4.1.0 adds private organizations, authenticated teams, role-based membership,
time-limited invitation plans, shared team project spaces, optimistic collaborative
revisions, immutable activity records, and portable team exports.

The implementation uses private WordPress record types for hosted operation and retains
browser-local and offline fallbacks. No paid external database is required for the
baseline deployment.

The unified selector expands from 22 to 23 studios with the new **Team Workspace**.
