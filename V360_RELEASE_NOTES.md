# Sustainable Catalyst Workbench v3.6.0

## Computational Intelligence and Predictive Analytics

This release adds a local-first, auditable predictive-analytics studio to the unified Workbench. It supports dataset profiling, split and feature planning, regression and classification metrics, cross-validation summaries, transparent linear-trend forecasting, drift and leakage audits, subgroup performance checks, model cards, cross-language reproducibility plans, and experiment scaffolds.

The baseline implementation does not train arbitrary remote models and does not automate high-stakes decisions. Private WordPress records are available to authenticated editors; anonymous use remains browser-local.
