Workbench v1.5.0 — Exportable Calculation Reports

1. Push the updated repository package:

cd ~/Downloads
chmod +x push_workbench_v1_5_0_exportable_reports_to_github.sh
./push_workbench_v1_5_0_exportable_reports_to_github.sh

2. Redeploy the Workbench Render service:

Root Directory: backend
Build Command: pip install --upgrade pip && pip install -r requirements.txt
Start Command: ./start.sh
PYTHON_VERSION=3.12.11

3. Confirm backend health:

https://YOUR-WORKBENCH-SERVICE.onrender.com/health

Expected version: 1.5.0

4. Confirm report endpoint after running a calculator:

POST /reports/calculation

5. Upload the WordPress plugin zip:

sustainable-catalyst-workbench-plugin-v1.5.0.zip

6. Test in WordPress:

- Open Workbench.
- Run Core Engineering Calculators.
- Click Markdown Report, HTML Report, Copy Report, or Print/PDF.
