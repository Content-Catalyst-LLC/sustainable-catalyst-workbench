# Workbench v4.0.0 — Connected Scientific and Engineering Workbench

Version 4.0.0 is the first connected production milestone for Sustainable
Catalyst Workbench. It adds a canonical project contract and orchestration layer
across all existing studios and the broader Sustainable Catalyst platform.

## New connected records

- Capability registry
- Connected project contract
- Cross-record connection graph
- Dependency-aware workflow plan
- Shared variables, assumptions, units, evidence, and uncertainty context
- Required and optional integration-health report
- Requirements-to-record/evidence/review traceability matrix
- Conflict-aware synchronization plan
- Connected technical dossier
- Human-controlled connected-release manifest
- Extension-contract validation report

## Platform connections

- Sustainable Catalyst Workbench
- Sustainable Catalyst Lab
- Site Intelligence
- Decision Studio
- Research Librarian
- Knowledge Library
- Browser-local and private WordPress storage
- Offline loopback FastAPI installation

## Safety and governance

The connected layer does not execute arbitrary commands, open remote shells,
publish automatically, control devices automatically, or claim professional
certification. Existing review, consent, backup, recovery, and public-release
gates remain authoritative.
