# R Statistics Extension

Use this extension when the Workbench needs advanced statistics, psychometrics, survey analysis, behavioral research methods, econometrics, or social-science modeling.

Recommended role:

```text
R = statistical specialist
```

Possible Workbench tools:

- Behavioral Research Methods analyzer
- Organizational survey scoring
- Psychometric reliability calculator
- Policy evaluation regression tool
- Sustainability indicator dashboard

Suggested API approach:

- Expose R tools through plumber.
- Call the R service from the Python/FastAPI backend.
- Keep final user-facing explanation in the Python/AI orchestration layer.
